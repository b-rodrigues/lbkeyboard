# lbkeyboard: Keyboard Layout Analysis and Optimization

Tools for analyzing and optimizing keyboard layouts for Luxembourg and
multilingual typing. Includes layout visualization, genetic algorithm
optimization with a Carpalx-inspired effort model, and a composable
rules system for customizing optimization constraints.

## Layout Visualization

- [`ggkeyboard`](https://b-rodrigues.github.io/lbkeyboard/reference/ggkeyboard.md):
  Plot keyboard layouts

- [`heatmapize`](https://b-rodrigues.github.io/lbkeyboard/reference/heatmapize.md):
  Create frequency heatmaps

- [`print_layout`](https://b-rodrigues.github.io/lbkeyboard/reference/print_layout.md):
  ASCII keyboard visualization

- [`plot_layout`](https://b-rodrigues.github.io/lbkeyboard/reference/plot_layout.md):
  ggkeyboard wrapper for optimized layouts

## Optimization

- [`optimize_layout`](https://b-rodrigues.github.io/lbkeyboard/reference/optimize_layout.md):
  Genetic algorithm optimization

- [`calculate_layout_effort`](https://b-rodrigues.github.io/lbkeyboard/reference/calculate_layout_effort.md):
  Calculate typing effort

- [`compare_layouts`](https://b-rodrigues.github.io/lbkeyboard/reference/compare_layouts.md):
  Compare multiple layouts

## Rules System

- [`fix_keys`](https://b-rodrigues.github.io/lbkeyboard/reference/fix_keys.md):
  Fix keys in place (hard constraint)

- [`prefer_hand`](https://b-rodrigues.github.io/lbkeyboard/reference/prefer_hand.md):
  Soft hand preference

- [`prefer_row`](https://b-rodrigues.github.io/lbkeyboard/reference/prefer_row.md):
  Soft row preference

- [`balance_hands`](https://b-rodrigues.github.io/lbkeyboard/reference/balance_hands.md):
  Hand balance preference

- [`keep_like`](https://b-rodrigues.github.io/lbkeyboard/reference/keep_like.md):
  Match reference layout
